#Owl links App
=================================

Owl link (em tradução livre Coruja links) é um projeto open source para compartilhamento e distribuição de links.

O projeto foi escrito utilizando o [Play Framework 2.3.x](https://www.playframework.com/) e Java 8 entre outras bibliotecas Java.

Estão realcionados a esse projeto um API - [API Owl Links](https://github.com/johnidm/owl-links-api), uma versão mobile [Owl Links Mobile](https://github.com/johnidm/owl-links-mobile) e um [Dashboard] (https://github.com/johnidm/owl-links-dashboard).
